夕语（Vesper）—— 你的本地 AI 桌面伴侣
========================================

夕语是一个开源的本地 AI 桌面聊天应用。它运行在你的 Windows 电脑上，
无需云端服务（除 AI 模型 API 外），所有对话数据完全由你掌控。


## 特性

  ● 本地运行 —— 下载即用，无需安装任何环境
  ● 流式对话 —— AI 回复逐句呈现，像真人聊天一样自然
  ● 三层记忆 —— 短期上下文 + 对话摘要 + 向量检索，AI 记得你说过什么
  ● 智能问候 —— 对话尾巴 + 时间关怀 + 主动交互
  ● 天气关怀 —— 每天三次天气推送，聊天卡片 + 系统通知
  ● 情感系统 —— AI 情感追踪，好感度/信任度，9 种情绪状态
  ● 四个主题 —— 暗色、亮色、樱花粉、夕语星光
  ● 工具集成 —— 天气查询、IP 定位、联网搜索、待办、笔记、提醒
  ● 系统托盘 —— 关闭窗口后后台运行，双击托盘恢复
  ● 数据可移植 —— 一键导出/导入全部聊天记录
  ● 离线可用 —— 除 AI 模型 API 外，所有功能本地运行


## 技术栈

  ● 后端：Python 3.10 + FastAPI + SQLite + ChromaDB 向量数据库
  ● 前端：Vue 3 + Vite
  ● 桌面壳：pywebview (Edge WebView2)
  ● AI 模型：DeepSeek Chat API（可替换为任何 OpenAI 兼容接口）
  ● 嵌入模型：paraphrase-multilingual-MiniLM-L12-v2（本地运行）
  ● 天气服务：Open-Meteo（免费，无需 API Key）


## 快速开始

  当前版本: v0.4.0 (2026-05-23)
  代码版本: 5.0.0

  1. 下载 Vesper.exe
  2. 双击运行
  3. 在设置中填入 API Key（DeepSeek / OpenAI / 任意兼容接口）
  4. 开始聊天

## 配置

  编辑 config.json 或通过设置面板修改：

  {
    "api_key": "",           // API Key
    "api_base_url": "...",   // API 地址
    "api_model": "...",      // 模型名称
    "personality": {         // AI 性格
      "tone": "冷静",        // 语气：冷静/活泼/温柔/毒舌/傲娇
      "length": "短"         // 回复长度：极短/短/中等/长/详细
    },
    "custom_system_prompt": ""  // 自定义人设（留空使用默认）
  }


## 主题

  ● 暗色 —— 深色背景，护眼
  ● 亮色 —— 浅色背景，清爽
  ● 樱花粉 —— 粉色系，温暖
  ● 夕语 —— 青紫星光，神秘优雅


## 系统托盘

  ● 关闭窗口 → 最小化到托盘，后端继续运行
  ● 双击托盘图标 → 恢复窗口
  ● 右键托盘 → 显示窗口 / 退出
  ● 再次启动 → 自动激活已有窗口
