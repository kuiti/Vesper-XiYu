# Vesper-XiYu（夕语）

开源本地 AI 角色扮演聊天引擎。每个角色拥有独立记忆、情感和人格，像真人一样长期陪伴。MIT 开源。

---

## 仓库结构

| 文件夹 | 用途 |
|--------|------|
| [`vesper_backend/`](vesper_backend/) | 后端核心：FastAPI 引擎 + 桌面启动器 + 前端构建产物，项目主入口 |
| [`vesper_frontend/`](vesper_frontend/) | Vue 3 前端源码（当前使用） |
| [`vesper_new_frontend/`](vesper_new_frontend/) | 新版前端（开发中） |
| [`vesper_mobile/`](vesper_mobile/) | 移动端适配 |

---

## 快速开始

进入主目录：

```bash
cd vesper_backend
```

详见 [`vesper_backend/README.md`](vesper_backend/README.md)。
