# 夕语（Vesper）v0.3.0

> 一个隐私优先的本地 AI 桌面伙伴

---

## 系统要求

| 项目 | 最低要求 |
|------|----------|
| 操作系统 | Windows 10/11（64位） |
| 内存 | 4GB（推荐 8GB） |
| 磁盘空间 | 500MB（EXE 版）/ 200MB（源码版） |
| 网络 | 需要（AI API 调用 + 天气查询） |
| 运行时 | WebView2（Windows 10/11 自带） |

---

## 安装方式

### 方式一：EXE 一键运行（推荐）

1. 下载 `Vesper.exe`
2. 双击运行
3. 首次启动会自动创建数据目录
4. 在设置中配置 API Key
5. 开始聊天

### 方式二：源码运行

1. 确保已安装 Python 3.10+
2. 下载源码并解压
3. 打开终端，进入项目目录
4. 创建虚拟环境：
   ```bash
   python -m venv venv
   ```
5. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
6. 启动程序：
   ```bash
   python launcher.py
   ```

## 配置说明

### API 设置

1. 点击右上角齿轮图标打开设置
2. 在「API」标签页配置：
   - **API Key**：你的 AI 服务密钥（如 DeepSeek、OpenAI）
   - **API 地址**：默认 `https://api.deepseek.com/v1`
   - **模型**：默认 `deepseek-chat`

### 天气关怀

1. 在设置「通知」标签页开启「每日天气推送」
2. 配置城市（设置 → API → 城市）
3. 每天 7:00、12:00、19:00 自动推送天气

### 主题切换

设置 → 外观 → 主题：
- 🌙 **暗色**：深色背景，护眼
- ☀ **亮色**：浅色背景，清爽
- 🌸 **樱花粉**：粉色系，温暖
- ✨ **夕语**：青紫星光，神秘优雅

### 系统托盘

- 关闭窗口 → 最小化到托盘，后端继续运行
- 双击托盘图标 → 恢复窗口
- 右键托盘 → 显示窗口 / 退出
- 设置中可关闭「关闭时托盘提示」

## 常见问题

**Q: 启动后白屏？**
A: 确保 WebView2 已安装。Windows 10/11 通常自带，如缺失可从微软官网下载。

**Q: API 连接失败？**
A: 检查 API Key 是否正确，网络是否正常。可在设置中测试连接。

**Q: 天气不推送？**
A: 确保已在设置中开启「每日天气推送」，并配置了正确的城市。

**Q: 如何完全退出？**
A: 右键系统托盘图标 → 退出。或在设置中关闭托盘行为。
