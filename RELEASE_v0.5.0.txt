Vesper（夕语） v0.5.0 Release Notes

概述

v0.5.0 是一次大规模质量更新。经过四轮全量代码审计（覆盖 89 个源文件、全部 API 端点、Core 模块、前端组件、WebSocket 协议），共修复 24 个 Bug，新增多项体验优化。致敬"佐仓"。

代码版本: 5.0.0
发布日期: 2026-05-25



崩溃级修复 (5 项)

问题  影响
  
FTS5 全文搜索 `SELECT role, timestamp FROM chat_fts` 列不存在  所有聊天搜索请求返回 HTTP 500
`init_db()` 从未被调用，schedule/achievements/ai_diary 表缺失  日程/成就/AI日记功能崩溃
向量重建时 numpy 数组真值判断 `if embeddings:` 报错  重建向量索引中途崩溃，HNSW 索引为空
LockScreen `maxlength="4"` 硬编码  设置 6 位 PIN 的用户永久锁死无法解锁
天气推送 `last_push_date` 只记日期不记小时  每日仅推送一次（7:00），12:00 和 19:00 被跳过

高风险修复 (8 项)

问题  影响
  
`recall_rule` 计算了但从未拼入 `final_prompt`  记忆回忆功能完全无效
`personality.get("recall_past")` key 名与存储不一致  设置面板"回忆过去"选项无法生效
关系系统情感状态机 `or` 应为 `and`  AI 情感大面积误判（cold/hurt/annoyed 分支不可达）
高德 API Key 设置页永远不回显  前端读 `s.amap_key`，后端返回 `has_amap_key`
云同步 passphrase 明文存入 config 表  加密密码直接泄露
全量备份导出含 `api_key` 明文  备份文件直接泄露 API 密钥
系统通知开关 `use_system_notification` 从未被读取  关闭开关后仍然弹通知
WebView2 geolocation 每次启动重弹权限  未预授权地理位置

中等修复 (8 项)

问题  影响
  
safeLinkify 双引号未转义  XSS 漏洞：URL 可注入 onclick 属性
WeatherCard `wind_speed` 为 null 时渲染出字面 "null"  风向/风速显示异常
意图检测 "最近怎么样" 误匹配新闻搜索  日常问候触发不相关的网络搜索
TTS `genie.tts()` 在 async 协程内同步阻塞  并发 TTS 请求时事件循环无响应
`saveCfg` 无防抖  快速切换设置开关导致前后端状态不一致
逐字模式 `last.content = s` 覆盖已输入的字符  typewriter 第一句显示不完全
逐字模式打字动画点与消息气泡并存  底部出现多余空气泡
自定义 system prompt 跳过 `length_rule`  AI 输出失控（不受长度约束）

体验优化 (3 项)

改动  说明
  
好感度/信任度双向牵制系统  信任越高好感涨得越快，反之亦然。7 档渐进乘数互相拉动
性格演化门槛大幅降低  从"月"级别缩短到"天"级别，聊几天就能看到性格数据变化
主动消息融入天气和位置  问候时会自然提及当地天气，不再只是"回来啦"

新增功能

- 通知风格扩展为 6 种（温暖/随意/幽默/简洁/傲娇/自由）
- 语气新增"自由"选项（AI 根据情境自行调整）
- Notes/Countdowns 新增 PATCH 编辑端点
- 目标系统新增创建/删除端点
- 地理位置预授权覆盖全部端口
- 兜底问候语扩充为每时段 5 条随机选取
- LLM 问候增加 3 次重试 + 30s 超时容错
- STT 音频上传 10MB 上限
- 聊天消息时间显示"3分钟前"/"昨天 14:30"等相对时间
- 迷你关系状态栏（聊天顶部显示好感度/信任度）
- 启动问候融入天气和时间
