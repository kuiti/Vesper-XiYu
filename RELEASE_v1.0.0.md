# Vesper v1.0.0 Release Notes

> 发布日期：2026-06-08

## 全新架构

### 5-in-1 后端融合

从 SillyTavern 等项目中学习并融合了以下六大能力：

1. **角色卡系统** —— 兼容 SillyTavern V3 标准格式，支持 JSON/PNG 导入导出，随时切换 AI 人设
2. **多 LLM 后端抽象** —— 统一接口支持 DeepSeek / OpenAI / Ollama 等，一键切换
3. **向量记忆** —— ChromaDB + sentence-transformers 语义搜索，记忆召回更精准
4. **提示词管道引擎** —— 20 种可组合的 PromptPipe，取代 700 行硬编码
5. **安全层** —— SSRF 防护、SQL 注入检测、路径穿越防护、输入清洗

### 全新前端面板

- 角色卡管理（导航栏新增「角色」入口）
- 用户画像（记忆页查看 AI 对你的了解）
- 云端同步（设置页配置 WebDAV 加密备份）

### 修复清单

**严重级：**
- 聊天记录删除时未清理句子索引，导致 RAG 返回已删内容的"幽灵"问题
- 情感演化直接 SQL 写 config 表绕过缓存，导致配置不一致
- 角色卡路由前缀为 `/api/characters` 与其他路由不统一，前端无法调用

**中等级：**
- 完全重置遗漏 5 张关联表（entities, memory_history 等）
- 删除单条消息触发全量向量重建，大数据量下性能灾难
- ai_background JSON 两处独立解析，不一致风险

**其他：**
- 向量模型下载指向国内镜像（hf-mirror.com），解决首次安装失败
- 重启后关系缓存未清除（Cline 审计发现并修复）
- 配置文件读写全面收敛到 get_config/set_config
- 多个函数内延迟导入路径修正

## 升级说明

直接从 v0.6.0 升级：下载新版解压覆盖即可。`data/` 目录中的数据（聊天记录、设置）不会丢失。
