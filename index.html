她不是工具，是记得你的人

长期记忆

记住你说过的话——不是每次重置的金鱼。存原文、建索引，不确定的事会说不确定。

人设锚定

角色性格随互动自然演化。可以成长，但不会变成另一个人。

共情增强

感觉到你在敷衍时换个话题，连续低落时安静陪着。懂什么时候该说、什么时候该听。

主动关怀

在你沉默时问候，融入天气和位置。深夜不打扰。

有温度的工具

待办、笔记、提醒、倒计时——不是冷冰冰的功能列表，带着记忆和情感。

绝对隐私

所有数据存储在本地，不上传任何服务器。可随时导出或删除。

快速开始

下载解压

从 Releases 下载打包版，解压到任意目录

获取 API Key

推荐 DeepSeek（国内可用，新用户送额度）。注册后在 API Keys 页面创建 Key。

配置

用记事本打开 config.json，在 api_key 字段填入你的 Key。

启动

双击 Vesper.exe 开始聊天。

也支持 源码运行（Python 3.11+）

夕语从「佐仓」项目中脱颖而出，致敬。
