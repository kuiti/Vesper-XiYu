## Purpose and Scope

The Chat Summarization Extension (internally managed as the Memory Extension) provides a mechanism for maintaining long-term context by condensing previous chat history into a persistent summary. This summary is then injected back into the prompt, allowing AI models to retain core facts and plot points after the original messages have rotated out of the active context window.

This document details the extension's architecture, summarization strategies (Extras, Main API, and WebLLM), prompt injection logic, and the technical implementation of rolling memory.

## System Overview

The extension operates as a background worker that monitors chat state and triggers summarization based on configurable thresholds (message count or word count). It hooks into the SillyTavern event system to react to chat loads and new message generation.

## Summarization Strategies

The extension supports three primary methods for generating summaries, each with distinct prompt construction logic and token management.

### 1. Main API (Standard)

This source uses the primary AI connection configured in SillyTavern. It supports three prompt_builder modes:

- Classic (0): Uses generateQuietPrompt(). This follows the standard prompt assembly pipeline, including World Info and Author's Notes (unless SkipWIAN is enabled). public/scripts/extensions/memory/index.js681-753 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L681-L753)

- Raw Blocking (1): Uses getRawSummaryPrompt() to build a simplified message list. It calls deactivateSendButtons() to prevent user input during the process. public/scripts/extensions/memory/index.js696-704 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L696-L704)

- Raw Non-Blocking (2): Similar to Raw Blocking but allows the user to continue typing while the summary generates by utilizing generateRaw() without UI lock. public/scripts/extensions/memory/index.js706-714 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L706-L714)

### 2. WebLLM Extension

Uses the local browser-based LLM via the WebLLM extension. It utilizes generateWebLlmChatPrompt() and specific token counting via countWebLlmTokens(). This allows summarization to happen entirely on the client side without hitting the main API. public/scripts/extensions/memory/index.js628-679 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L628-L679) public/scripts/extensions/memory/index.js49-52 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L49-L52)

### 3. Extras API (Deprecated)

Communicates with the SillyTavern-extras server. It builds a text buffer by reversing the chat history until the getSourceContextSize() limit is reached. public/scripts/extensions/memory/index.js827-891 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L827-L891)

## Trigger Logic and Thresholds

Summarization is not performed on every message. Instead, getSummaryPromptForNow() evaluates the chat history to see if an update is required based on the configured extension_settings.memory.

### Update Conditions

- Interval: Controlled by promptInterval. Triggers when messagesSinceLastSummary >= promptInterval. public/scripts/extensions/memory/index.js589-591 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L589-L591)

- Word Count: Controlled by promptForceWords. Triggers if the word count since the last summary exceeds this value. public/scripts/extensions/memory/index.js605-607 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L605-L607)

- Manual: Triggered via the "Summarize now" UI button (#memory_force_summarize) or the /summarize slash command. public/scripts/extensions/memory/index.js502-539 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L502-L539) public/scripts/extensions/memory/settings.html30-33 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/settings.html#L30-L33)

### Raw Prompt Construction

When using "Raw" builders, the function getRawSummaryPrompt() iterates backward from the current message to the last message that contains a summary. It concatenates the intervening messages into a formatted string: {{name}}: {{message}}, stripping reasoning blocks via removeReasoningFromString(). public/scripts/extensions/memory/index.js755-825 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L755-L825)

## Prompt Injection and Formatting

Once a summary is generated, it is processed through a template and injected into the final prompt using the standard extension prompt system.

### Injection Pipeline

- Formatting: formatMemoryValue() applies the extension_settings.memory.template. Default: [Summary: {{summary}}]. public/scripts/extensions/memory/index.js77-89 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L77-L89)

- Injection: setExtensionPrompt() is called with the following parameters:

- identifier: MODULE_NAME (1_memory).

- value: The formatted summary.

- position: extension_prompt_types (e.g., IN_PROMPT, IN_CHAT, BEFORE_PROMPT).

- depth: The insertion depth for IN_CHAT mode.

- role: The extension_prompt_roles (System, User, or Assistant).

## Configuration and UI

The extension settings are managed via the loadSettings() function, which synchronizes the extension_settings.memory object with the DOM elements in the settings panel defined in settings.html.

### Key Settings

- memoryFrozen: Disables automatic updates. While paused, the summary remains static. public/scripts/extensions/memory/index.js109 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L109-L109) public/scripts/extensions/memory/settings.html34 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/settings.html#L34-L34)

- SkipWIAN: If true, World Info and Author's Notes are stripped from the text sent to the summarizer (Main API only). public/scripts/extensions/memory/index.js110 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L110-L110) public/scripts/extensions/memory/settings.html35-38 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/settings.html#L35-L38)

- prompt_builder: Determines if the summarizer uses the full prompt pipeline (Classic) or a "Raw" message list. public/scripts/extensions/memory/index.js138 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L138-L138) public/scripts/extensions/memory/settings.html48-62 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/settings.html#L48-L62)

### Slash Commands

The extension registers several commands via SlashCommandParser to allow scriptable control over memory.

- /summarize: Forces a summarization of the current chat or provided text. public/scripts/extensions/memory/index.js502-539 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L502-L539)

- /summary-restore: Reverts to the previous summary by deleting the current one from the chat metadata. public/scripts/extensions/memory/index.js973-1004 (https://github.com/SillyTavern/SillyTavern/blob/51ad27fb/public/scripts/extensions/memory/index.js#L973-L1004)
