    """辩证结论注入（每10条注入一次）"""
    zone = "dynamic"
    def process(self, ctx):
        if not ctx.is_module_enabled("conclusion"):
            return None
        if ctx.msg_counter % 10 != 0:
            return None
        from core.conclusion_engine import get_conclusion_context
        return get_conclusion_context()

## 任务 C1A：打字动画 Blip（frontend）

### 基础实现（MiMo 做）

**目标**：AI 回复逐字显示，而不是一次性冒出来

文件：`sakura_frontend/src/components/ChatBubble.vue`

**实现思路**：
1. 在 `ChatBubble.vue` 中新增 `typewriter` 模式
2. 收到完整的 AI 回复后不直接显示，而是启动定时器逐个字符追加
3. 支持速度控制（快/慢/正常）
4. 重要/情感词变色（高兴→蓝色，生气→红色）

```vue
// ChatBubble.vue 新增逻辑
data() {
  return {
    displayText: '',
    typewriterSpeed: 30,  // ms per char
    typewriterTimer: null,
    isTyping: false,
  }
},
methods: {
  startTypewriter(fullText) {
    this.displayText = ''
    this.isTyping = true
    let i = 0
    this.typewriterTimer = setInterval(() => {
      if (i < fullText.length) {
        this.displayText += fullText[i]
        i++
      } else {
        clearInterval(this.typewriterTimer)
        this.isTyping = false
      }
    }, this.typewriterSpeed)
  },
  // 情感词变色函数
  highlightEmotion(text) {
    const emotionWords = {
      '高兴': 'color: #4CAF50',
      '开心': 'color: #4CAF50',
      '难过': 'color: #2196F3',
      '生气': 'color: #f44336',
      '爱你': 'color: #E91E63',
    }
    // 用 v-html 渲染带颜色的文本
    let result = text
    for (const [word, style] of Object.entries(emotionWords)) {
      result = result.replaceAll(word, `<span style="${style}">${word}</span>`)
    }
    return result
  }
}
```

**涉及文件**：
- `sakura_frontend/src/components/ChatBubble.vue`（~80行）

**验收标准**：
- AI 回复逐字出现，不是一次性显示
- 开关/速度调节正常
- 情感词颜色正确

### 优化（Claude 做）
