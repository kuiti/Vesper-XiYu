# 夕语进化计划
## 从聊天助手 → 有性格的陪伴 AI

> 对标参考：SillyTavern（角色扮演框架）、NemoLore（智能记忆）、Chapterizer（分章叙事）
> 插件参考：st-memory-enhancement（记忆表格）、ST-BME（记忆图谱）、JS-Slash-Runner（插件系统）
> 同源项目：[未婉进化计划](./未婉进化计划.md) — 共享核心层，夕语交付后再加工具线
> 起始状态：FastAPI + SQLite + ChromaDB + Vue 3，46 API + 36 Core 模块
> 最后更新：2026-06-22

---

## 架构说明

```
夕语 V1（聊天陪伴）
├── 核心层（所有用户共用）
│   ├── LLM 通信（llm_client + llm_provider）
│   ├── 记忆系统（profile + episodic + vector + summary）
│   ├── 关系系统（relationship）
│   ├── Auth + DB + 文件操作
│   └── 多模型路由（可选）
│
├── 聊天层（夕语专属）
│   ├── C1 沉浸式对话
│   ├── C2 角色人格深化
│   ├── C3 对话趣味性
│   ├── C4 情感连接
│   └── C5 记忆叙事
│
└── 前端（Vue 3）
    ├── 聊天界面
    ├── 时间线 UI
    ├── 表情/动画
    └── 记忆可视化（可选）

未婉 V2（工具助手）— 后期安装，共享核心层，加工具侧 prompt + API
```

## 交付顺序（分 4 批）

### 第一批：记忆基础（2 周）
```
A1A 辩证结论 → 用户感知"夕语真的懂我"
A1B 摘要增强 → 长期记忆更完整
A2A 多信号检索 → 回忆更准
C1A Blip动画 → 第一眼就看得出不一样
C1D 时间线UI → 能看到"夕语记住了什么"
```

### 第二批：沉浸感（2 周）
```
C1B 表情系统 → 聊天有画面感
C1C 场景氛围 → 对话有场景
C4C 分离处理 → 久别重逢有温度
C4D 称呼系统 → 变成"专属"的
```

### 第三批：人格与情感（3 周）
```
C2A 人格一致 → 夕语不OOC
C2B 记忆人格 → 表达更贴心
C2C 关系深度 → 好感度影响互动
C4A 深度记忆 → 精准回忆
C4B 主动关怀 → 会主动找你
```

### 第四批：趣味与叙事（3 周）
```
C3A 小游戏 → 聊天能玩
C3B 彩蛋 → 惊喜感
C3C 接梗 → 更自然的互动
C3D Quick Reply → 快捷命令
C5A 分章 → 对话变成故事
C5B 闪回 → 电影感回忆
C5C 故事线 → 长线叙事
C5D 回忆报告 → 周/月总结
A7C 概念学习 → 学用户说话
```

### 第五批：记忆深度优化（3 周）
```
C6A 记忆表格 → 用户能看到和编辑所有记忆
C6B 双通道验证 → 向量召回准确率提升
C6C 图谱可视化 → 记忆关系一目了然
C6D 分层摘要 → 总结不丢信息
```

### 第六批：工具与扩展（持续）
```
T1 Prompt Inspector → 调试 prompt
T2 Token 计数器 → 感知成本
T3 代码执行 → 在聊天框里写代码
T4 插件系统 → 生态开放
T5 信息采集 → RSS/新闻
T6 推送通知 → 移动端推送
```
